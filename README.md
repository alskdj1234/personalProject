구현 하고 싶은 거 정리

1. 2팩 만원 자동화 프리셋 아이디어
내용: 마트에서 흔히 하는 '2팩 만원' 같은 묶음 할인 프로모션을 시스템이 자동으로 계산하도록 프리셋을 심는 기능.

핵심: 딸기, 포도처럼 서로 다른 상품을 골라도 같은 행사 상품이면 교차 선택이 인정되어 할인이 적용되는 비즈니스 로직.

2. 박스 바코드 오스캔 처리 아이디어
내용: 초보 알바들이 낱개(단품)를 팔거나 검수해야 할 때 실수로 박스 바코드를 찍는 문제를 해결하는 기능.

핵심: 무조건 결제를 막아버리는(Block) 게 아니라, 시스템이 박스 바코드임을 인지하고 "낱개 수량(예: 24개)으로 자동 전환해서 등록할지" 유연하게 물어보고 처리하는 예외 로직.

3. 원청 명세서 마스터 바코드 검수 아이디어 (ASN)
내용: 하나하나 띡띡 찍으며 검수하는 대신, 원청(공급사)이 명세서에 심어 보낸 '마스터 바코드'를 찍어 한방에 실사 비교하는 아이디어.

핵심: 바코드를 스캔하면 입고 예정 상품 목록과 수량이 터미널 화면에 쫙 연동되고, 알바는 실물 개수만 확인해 정합성을 체크하는 최종 진화형 시스템.
